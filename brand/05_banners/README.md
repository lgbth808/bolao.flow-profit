# Banners

Guarda imagens de banner por canal e contexto.

Subpastas:

- `hero/`: banners principais.
- `login/`: fundos ou chamadas para login.
- `admin/`: fundos ou imagens administrativas.
- `public/`: pagina publica dos palpiteiros.
- `whatsapp/`: imagens para disparos ou previews.
- `instagram/`: pecas para Instagram.
- `facebook/`: pecas para Facebook.

Manter proporcao e destino do banner documentados no nome do arquivo.
