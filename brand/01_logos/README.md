# Logos

Guarda as variacoes oficiais de logotipo do **Bet Barao by d. Rosa**.

Subpastas:

- `principal/`: logo principal para telas de alto impacto.
- `horizontal/`: versoes horizontais para cabecalhos e areas largas.
- `simbolo/`: marca reduzida, icone ou monograma.
- `texto/`: assinatura tipografica.
- `dark/`: versoes para fundos escuros.
- `svg/`: versoes vetoriais finais.

Nao colocar rascunhos ou versoes nao aprovadas nesta pasta.
