# Personagem d. Rosa

Guarda a personagem d. Rosa, que e o rosto principal da identidade.

Subpastas:

- `avatar/`: rosto e recortes para perfil.
- `expressoes/`: variacoes de humor e reacao.
- `poses/`: poses tematicas para interface e marketing.
- `corpo_inteiro/`: ilustracoes completas.
- `meio_corpo/`: recortes de meio corpo.
- `cafe_da_rosa/`: versoes ligadas ao tema Cafe da Rosa.

Nao usar mascote masculino como personagem principal.
