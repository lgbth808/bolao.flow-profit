# Editaveis

Guarda arquivos-fonte para edicao profissional.

Subpastas:

- `svg/`: vetores editaveis.
- `ai/`: Adobe Illustrator.
- `figma/`: arquivos ou exports do Figma.
- `pdf/`: PDFs editaveis ou prontos para grafica.

Nao usar estes arquivos diretamente no app sem exportar para formato adequado.
