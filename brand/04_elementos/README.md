# Elementos Graficos

Guarda elementos visuais de apoio da identidade.

Subpastas:

- `bola/`: elementos de futebol.
- `trofeu/`: premios e trofeus.
- `cafe_da_rosa/`: elementos ligados ao Cafe da Rosa.
- `rua_barao/`: elementos da Rua Barao.
- `badges/`: selos e marcadores.
- `decoracoes/`: ornamentos, padroes e detalhes visuais.

Use elementos com moderacao para manter a interface legivel.
