# Tutoriais

Guarda assets e roteiros visuais para ensinar o uso do bolao.

Subpastas:

- `expectativa/`: orientacao antes do jogo.
- `aprenda/`: passo a passo de uso.
- `regras/`: explicacoes de regras.
- `pagamento/`: fluxo de PIX e baixa.
- `premiacao/`: rateio e pagamento de premios.
- `resultados/`: placares e vencedores.

Os materiais devem ser simples e claros para usuarios 60+.
