# Bet Barao by d. Rosa Brand Kit

Este diretorio organiza os ativos oficiais da identidade visual do projeto **Bet Barao by d. Rosa**.

Use esta area para guardar logos, personagem d. Rosa, icones, elementos graficos, banners, tutoriais, pecas de marketing, arquivos editaveis e documentacao da marca.

Regras:

- Nao salvar arquivos falsos ou provisórios como se fossem finais.
- Manter nomes claros, versionados e sem acentos quando forem assets.
- Separar arquivos de producao dos arquivos editaveis.
- Preferir `png`, `svg` e fontes editaveis nos diretorios apropriados.
- Registrar decisoes de identidade em `09_documentacao/`.
